# AI Configurations

This repository contains custom instruction sets for AI tools used in my development workflows.

## Purpose

Provides standardized guidelines and conventions for two types of AI interactions:

- **Requirements LLM** (ChatGPT): Handles requirements gathering, analysis, and documentation through direct LLM conversation
- **Coding Agent** (Cline/Claude): Handles code implementation, testing, and technical execution through agentic workflows

## Overview

Each configuration has specific instructions that define behavior, output format, and operational standards. These configurations ensure consistent, professional, and implementation-focused results across projects.

- The **requirements-llm.md** file contains instructions for LLMs (like ChatGPT) to guide requirements analysis through conversational interaction
- The **coding-agent.md** file contains instructions for coding agents (like Cline/Claude) to guide autonomous code implementation

The instruction files serve as system prompts that guide AI responses and decision-making throughout the development process.
